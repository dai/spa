spa by dejanstrbac

Single Page App / javascript

SPA is a lightweight jQuery plugin that allows for DVC (Data-View-Controller) that are read/browse heavy. Most common use case is in ecommerce where the catalog data does not require often reloads. SPA has been inspired by Amazon's MyHabit flash sales website.
